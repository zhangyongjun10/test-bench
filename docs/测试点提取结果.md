# Agent 验证平台 - 测试点提取结果

---

## 功能模块1：Agent 生命周期管理

### 正常流程测试点：
- 【TC-001】Agent 创建：通过表单输入 Agent 名称、描述、baseURL、APIKey 创建 Agent
- 【TC-002】APIKey 加密存储：验证 APIKey 敏感信息加密存储到数据库
- 【TC-003】Agent 连接测试：创建/编辑时测试连接验证 Agent 是否可正常调用
- 【TC-004】Agent 配置保存：验证新建 Agent 配置正确保存到数据库
- 【TC-005】Agent 信息编辑：编辑已注册 Agent 的所有字段信息
- 【TC-006】编辑后重新测试：编辑后支持重新测试连接验证配置正确性
- 【TC-007】编辑更新保存：修改后正确更新保存 Agent 配置
- 【TC-008】Agent 删除：删除不需要的 Agent，删除前显示二次确认
- 【TC-009】关联数据级联删除：删除 Agent 后同时清理关联的所有测试结果
- 【TC-010】Agent 列表展示：查看所有已注册 Agent 列表，展示名称、创建时间、最后测试时间
- 【TC-011】按名称搜索过滤：按关键词搜索过滤 Agent 列表
- 【TC-012】进入详情页：点击列表项跳转到 Agent 详情页

### 异常&边界值测试点：
- 【TC-013】必填项为空创建：Agent 名称或 baseURL 为空时，提示错误不创建
- 【TC-014】无效 baseURL 测试：baseURL 格式错误或不可达，测试连接返回失败提示
- 【TC-015】重复名称创建：同一名称多次创建，系统正确处理

- 【TC-023】非默认模型删除：删除非默认的模型配置

## 功能模块5：可观测性数据集成

### 正常流程测试点：
- 【TC-052】ClickHouse 连接配置：配置 Opik/Langfuse ClickHouse 连接地址、账号密码
- 【TC-053】ClickHouse 连接测试：支持测试连接验证配置正确性
- 【TC-054】数据源切换：支持切换 Opik 或 Langfuse 数据源
- 【TC-055】Trace 数据查询：根据 trace_id 查询所有 span 数据
- 【TC-056】Span 指标解析：解析 span 提取 LLM/tool/runtime 各类指标
- 【TC-057】指标存储：存储解析后的指标数据
- 【TC-058】LLM 指标提取：提取 TTFT(首包延迟)、TPOT(每 token 延迟)、Token 数量统计
- 【TC-059】Runtime 指标提取：提取 CPU 使用率、内存占用、GC 次数/耗时
- 【TC-060】工具调用指标提取：提取工具调用次数、错误率、平均延迟
- 【TC-061】百分位数计算：计算 P50/P75/P90/P99 百分位数
- 【TC-062】时序数据存储：存储所有指标到时序数据库(Prometheus 兼容)

### 异常&边界值测试点：
- 【TC-063】ClickHouse 连接失败：ClickHouse 地址或凭证错误，连接失败给出明确提示
- 【TC-064】Trace 不存在：trace_id 在 ClickHouse 中不存在，正确处理并记录
- 【TC-065】空 Trace 数据：获取到空 Trace 数据，系统正确处理不崩溃
- 【TC-066】超大 Trace 数据：Trace 包含上千个 span，系统能够正确处理解析
- 【TC-067】Span 缺少指标字段：Span 缺少某些字段，系统跳过不报错，记录可用指标
